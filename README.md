# 🎓 University Registration Form

A simple and responsive HTML-based university registration form project.

---

## 📌 Overview
This project is a static web form created using HTML. It simulates a university admission registration system where users can enter their personal, contact, and academic details.

---

## ✨ Features
- Personal details input (Name, DOB, Gender)
- Contact information (Email, Phone, Address)
- Course selection dropdown
- File upload option for profile photo
- Extracurricular activities selection
- Password and security question fields
- Reset and Submit functionality

---

## 🛠️ Tech Stack
- HTML5
